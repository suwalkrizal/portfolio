django

Django is a high-level Python web framework that helps you build secure and scalable web applications quickly. It comes with built-in tools like authentication, admin panel, and ORM, making development faster and more organized.

Html

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines elements like headings, paragraphs, links, images, and more, forming the basic layout of a webpage.

CSS

CSS (Cascading Style Sheets) is used to style and design the layout of web pages created with HTML. It controls colors, fonts, spacing, and positioning to make websites look attractive and user-friendly.

Github

GitHub is a platform for storing and managing code using version control with Git. It allows developers to collaborate, track changes, and work on projects together from anywhere.

PostgreSQL

PostgreSQL is a powerful, open-source relational database system known for its reliability and advanced features. It supports complex queries, data integrity, and is widely used with web applications like those built with Django.

Restframework

Django REST framework (DRF) is a powerful toolkit for building Web APIs with Django. It simplifies tasks like serialization, authentication, and permissions, making it easy to create robust, secure, and scalable APIs.

OCR

OCR (Optical Character Recognition) is a technology that converts text from images, scanned documents, or handwritten notes into editable and searchable digital text. It's commonly used to automate data entry, digitize printed records, and enable text search in images.

Redis

Redis is an open-source, in-memory data store used as a database, cache, and message broker. It's known for its speed and is often used to store frequently accessed data like user sessions, real-time analytics, or queue jobs for background processing.

Waste Food Donation

College Project | Team Members: 2 | Connected via GitHub

Save2Serve is a food donation platform developed using Django for the backend and HTML, CSS, JavaScript for the frontend. The platform connects donors with people in need by collecting surplus food and distributing it efficiently. We implemented Dijkstra’s algorithm to calculate the shortest delivery route and K-Nearest Neighbors (KNN) algorithm to predict how many people can be served based on the donated food quantity. The project emphasizes teamwork, problem-solving, and the integration of algorithms into real-world use cases.
